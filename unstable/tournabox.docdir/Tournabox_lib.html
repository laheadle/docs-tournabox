The top-level tournabox library. Clients can use these types and values to find and run tournaboxes on web pages.

An HTML element that will contain a visible tournabox.

Get a list of all the containers on the page.

Given a container, return a tourney shell or None and an error message.

Play and display a tourney_shell.
