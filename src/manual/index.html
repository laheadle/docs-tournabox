Tournabox: The Manual

Imagine a small tournament called the "utopian open." You need three HTML elements to display it as a tournabox.

This is the element your readers will see, containing the tournament results. It looks like this:

The container must have class tournabox-container.

The id is used to find the entry and outcome elements (see below).

This element contains the names of all the entries (players or teams) in the tournament. Entries are listed in order: the first entry plays the second entry, the third entry plays the fourth entry, etc. In the second round, the winner of the match between #1 and #2 will play the winner of the match between #3 and #4, etc.

An entry looks like this:

NAME OPTIONAL-COUNTRY [OPTIONAL-SEED] (OPTIONAL ATTRIBUTE)

Where the optional country is a three letter country code, the optional seed is a number, and the optional attribute is a letter, such as 'w' for "withdrew" or 'q' for "qualifier."

The entries are found by starting from the id of the container, here "utopian-open," and adding the string "-entries." Thus, by giving the entries element the id "utopian-open-entries," you tell tournabox which element contains the entries for the container "utopian-open."

You should hide this element by using style="display: none" as in the example above.

This element contains the record of wins and losses. Each line lists the name of a winner.

The name can be abbreviated to save typing, as long as there is only one name matching the abbreviation. Abbreviations can come from anywhere in the name, and case does not matter. For example, to abbreviate "Joe Smith" you could enter "joe" or "smith".

Note that Sam Querrey (abbreviated 'qu') is listed twice. This means that he won in the first round, and then won again in the second round.

The outcomes are found by starting from the id of the container, here "utopian-open," and adding the string "-outcomes". Thus, by giving the outcomes element the id "utopian-open-outcomes," you tell tournabox which element contains the outcomes for the container "utopian-open."

Sometimes, you want to show the checkboxes -- the ways of grouping a tournabox -- in a different order. Other times, you don't want to show as many ways of grouping a tournabox. You can do either by adding the attribute tournabox-groups to your container element.

This attribute's content is the name and order of the grouping checkboxes you want to show.

You can also set the initial filter string using the attribute tournabox-filters.To give more than one filter string, separate each filter string by a comma, as in term 1,term2. Matches will be shown if they match any of the strings.
